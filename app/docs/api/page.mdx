# API Reference

## Модули

- **[Core](/docs/api/core)** — Основные классы для работы с оптическими симуляциями (8 классов, 1 функций)
- **[Elements](/docs/api/elements)** — Оптические элементы: линзы, апертуры, дифракционные слои, SLM и др. (13 классов)
- **[Networks](/docs/api/networks)** — Нейронные сети и оптические системы (4 классов)
- **[Specs](/docs/api/specs)** — Спецификации параметров для экспорта и сериализации (16 классов, 7 функций)
- **[Phase Retrieval Problem](/docs/api/phase_retrieval_problem)** — Алгоритмы восстановления фазы (3 классов, 5 функций)
- **[Visualization](/docs/api/visualization)** — Инструменты визуализации (3 классов, 6 функций)
