## Обзор компонентов

| Компонент | Описание |
|-----------|----------|
| `SimulationParameters` | Координатная система, длина волны, устройство |
| `Wavefront` | Комплексная амплитуда поля |
| `Element` | Базовый класс оптических элементов |
| `LinearOpticalSetup` | Последовательная оптическая система |
| `Detector` | Преобразование поля в интенсивность |
| `DetectorProcessorClf` | Классификация по областям детектора |
| `Clerk` | Логирование экспериментов |

## Рекомендуемый порядок изучения

1. **SimulationParameters** — основа всех расчётов
2. **Wavefront** — создание входных полей
3. **Elements** — оптические элементы
4. **Optical Systems** — сборка систем
5. **Detector** — измерение результата
6. **Optimization** — обучение систем
